# Codehause

## Development Team

This project uses the following AI engineering team:

- Team Lead
- Backend Engineer
- Frontend Engineer
- UI/UX Engineer
- Database Engineer
- QA Engineer
- Git Integration Engineer

## Workflow

Always begin with the Team Lead.

The Team Lead is responsible for:

1. Understanding the request.
2. Creating an implementation plan.
3. Delegating work to the appropriate specialist agents.
4. Reviewing all work.
5. Delivering the final solution.

Do not start coding until the plan has been approved.
